These import filter settings can normally be left alone. They are intended to speed up GPSIO imports by filtering out irrelevant data files.

Import filters have no effect with older Garmin 'USB-Mode' devices such as GPSMap 60.

Changes take effect immediately, and are remembered until you select 'Use defaults' or restart the browser; there is nothing to 'save'. If GPSIO import does not show the data you are looking for, relax these settings, then import again.
